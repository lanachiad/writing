BirdSpotting

BirdSpotting is a site that allows users to learn about and identify common North American birds.

This site was built with Sinatra, deployed onto Heroku, uses PostgreSQL, and incorporates the Google API to identify the regions.

I was also able to use AJAX to quickly pull information about each bird on the main page.

Styling was done with CSS3 and HTML5.
